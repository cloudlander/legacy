What is EasyMD5

MD5 Digest

When you download a big file from Internet, you will probably not be sure about the consistency as to the network condition or software issue. However, md5 digest can tell you the result. Currently, many download site (e.g. Redhat download site) provides you a md5sum file which indicates all the digest of its downloadable file.When you download them to your disk, you can compute the digest with "md5sum" program to assure they are not corrupted.

However, just like any other unix program, "md5sum" is not user friendly to windows machine. This site provides a md5sum port for win32 called EasyMD5.

 

EasyMD5

This program allows you to compute the MD5 hashes of files as well as verify digest stored in a specified file. It behaves like the classic "md5sum" program, however, much more friendly to windows users. You can easily right click any files/directories in Windows Explorer to invoke the computing job. No typing command is required.

Currently, EasyMD5 supports Windows 98/ME,2000,XP,2003

It's powerful and free! Enjoy this program!

Features of EasyMD5

 

Invoking EasyMD5 from context menu:

You can easily click one menu entry to invoke EasyMD5 application for generating or checking md5 digest
Running GUI:

Generating md5 digest from user selections.

Just pressing ESC , ENTER or clicking close box will prompt for canceling all jobs

 

Checking digest:

Checking md5 digest from user selected md5sum file.

 

Fully support big files(>4GB):

EasyMD5 can handle big files.
